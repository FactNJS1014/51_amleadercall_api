### API Get Data

#### 1. Get Work Order , Model Code , Model Name and lot size

```http
GET /vwork
```

#### 2. Get Record Data from AM_LDR_INFOHREC_TBL

```http
GET /info/record
```

#### 3. Get Record Data from AM_LDR_ACTIONHREC_TBL

```http
GET /action/record
```

#### 4. Get Record Data from AM_LDR_CONFIRMHREC_TBL

```http
GET /confirm/record
```
